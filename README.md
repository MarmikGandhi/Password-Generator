# Password Generator

A sleek and customizable password generator built using **HTML**, **CSS**, and **JavaScript**. This web application allows users to generate secure passwords with adjustable criteria including length, uppercase, lowercase, numbers, and symbols.

---

## Features

  - Customizable options for:
  - Password length (4 to 16 characters)
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Symbols
  - Copy to clipboard functionality
  - Real-time validation for length constraints
  - Visually appealing and responsive design
  - Mobile-friendly layout
